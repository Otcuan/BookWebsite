# Nhạc nền

Đặt file nhạc của bạn tại đúng đường dẫn:

```text
public/audio/background.mp3
```

Giữ tên file viết thường như trên, sau đó commit/push lên GitHub để Vercel deploy.
Nên dùng MP3 có bản quyền sử dụng phù hợp, bitrate khoảng 96–160 kbps và dung lượng
vừa phải để không tốn dữ liệu di động của bạn đọc.
